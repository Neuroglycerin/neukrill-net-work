  • Look at examples from different classes
  • Distribution of image dimensions (heatmap)
  • Class proportions
  • Image statistics (define more accurately when you get here)

Super Classes

We want a list of these, because then we can split up the above with some work. Think the only option is to write it by hand. There is a problem with this, because the classes don't all have their superclass in their name. Most of them do, but not all. Examples are some Shrimp-likes, others in crustaceans, heteropods and fecal pellets. So these superclasses are incomplete. To fix it we'd need to make a dictionary, with each of the classes sorted into the right corresponding superclasses.
